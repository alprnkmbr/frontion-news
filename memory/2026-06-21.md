### Lessons Learned
- When explaining technical steps to user, give CLEAR STEP-BY-STEP instructions:
  1. Click + button → add X
  2. Click + button → add Y
  3. Delete old Z
  4. Click arrow between A and B → set filter
  Don't assume user will figure out the order. List every step explicitly.
- **NEVER send manual LinkedIn posts or any manual action without explicit user approval first.** Cron handles everything automatically. Manual test posts caused duplicate errors and locked up Make.com.
- Test posts were sent before cron ran, causing the 08:00 cron to hit duplicate content errors → Make.com stopped the scenario. Don't repeat this mistake.
